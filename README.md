 **ARDUINO-UNO-UART-READER**
___________________



  
Modified version of the Justatic's UART bridge for Arduino Uno boards (<https://gist.github.com/Justasic/7d41d1af7bd6d825d2d3>) with a (mostly Vibe Coded) Spanish GUI that must be run with a python server (in localhost:5000).

--> run ``python3 install requirements.txt`` if something does not work :)
